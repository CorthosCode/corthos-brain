# \[Linux] Создание пользователя







## Создание пользователя с sudo правами

```shellscript
# Создать нового пользователя
sudo adduser username

# Добавить пользователя в группу sudo
sudo usermod -aG sudo username

# Проверить права
sudo -l -U username
```





***

## Команда `su`(switch user)

```shellscript
# Переключиться на другого пользователя (требует пароль целевого пользователя)
su - username

# Переключиться на root (требует пароль root)
su -

# Переключиться без создания новой среды (оставляет текущие переменные)
su username
```
