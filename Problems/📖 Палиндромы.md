
**Палиндром** - число, буквосочетание, слово, текст, одинаково читающееся в обоих направлениях.

## Определение является ли строка палиндромом:

```java
private boolean isPalindrome(String s, int start, int end) {
	while (start < end) {
		if (s.charAt(start) != s.charAt(end)) {
			return false;
		}
		start++;
		end--;
	}
	return true;
}
```


## Поиск всех палиндромов (O(N^3))

```java
for (int start = 0; start < s.length(); start++) {

		if (isPalindrome(s, start, end)) {
			// Палиндром найден
		}
	}
}
```

## Поиск всех палиндромов (O(N^2) - центр расширения)

Любой палиндром имеет центр:
- Для нечетной длины - один символ (`aba` -> `b` центр)
- Для четной длины - между двух символов (`abba` -> центр между `b` и `b`)

Всего `2n - 1` возможных центров:
- `n` - для нечетных (центр `i`)
- `n - 1` - для четных (центр между `i` и `i + 1`)

Алгоритм:
1. Представьте строку и каждую букву в ней рассматривайте как центр потенциального палиндрома.
2. Для каждого такого центра "расширяйтесь" в обе стороны - влево и вправо, сравнивая символы.
3. Пока символы слева и справа совпадают - расширяем палиндром.
4. Если встретились разные символы - прекращаем расширение.
5. Делаем для каждого возможного центра (четной и нечетной длины).

Вспомогательный метод расширения из центра:
